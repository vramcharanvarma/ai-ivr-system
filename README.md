# AI IVR System

An AI-powered Interactive Voice Response (IVR) system built using Python.

## Features

- Speech-to-Text using Whisper
- AI-generated responses using an LLM
- Text-to-Speech using gTTS
- Streamlit-based user interface

## Tech Stack

- Python
- OpenAI Whisper
- OpenAI API
- gTTS
- Streamlit

## Project Workflow

User Voice Input
↓
Speech-to-Text (Whisper)
↓
LLM Response Generation
↓
Text-to-Speech (gTTS)
↓
Audio Response to User


## System Highlights

- Implemented an end-to-end audio processing and response generation pipeline for seamless voice interactions.
- Optimized response flow to improve interaction quality and reduce conversational latency.
